# مجلد الصور (Images Folder)

قم بوضع صور المشروع الخاصة بك في هذا المجلد.

## كيفية الاستخدام:

1. ضع صورك هنا (مثلاً: `logo.png`, `hero.jpg`).
2. بما أن هذا المجلد داخل `public`، يمكنك الوصول للصور مباشرة باستخدام المسار `/images/اسم-الصورة`.

### مثال لتحديث الكود (data.ts):

بدلاً من استخدام روابط Unsplash الطويلة:
```typescript
items: [
  {
    id: 'rest',
    // ...
    // رابط صورة خارجي قديم
    // image: "https://images.unsplash.com/..."
    
    // الرابط الجديد من ملفاتك المحلية
    image: "/images/my-restaurant.jpg"
  }
]
```
